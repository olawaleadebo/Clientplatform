=====================================================
   QUICK FIX FOR CALL TRACKER 404 ERROR
=====================================================

ERROR: 
  Server responded with 404: path="/call-progress"

CAUSE: 
  Backend server is running old code without new endpoints

FIX: 
  RESTART THE BACKEND SERVER

=====================================================

WINDOWS USERS:
--------------
1. Press Ctrl+C in backend terminal (stop old server)
2. Run this command:
   cd backend && deno run --allow-all server.tsx

OR double-click:
   backend\RESTART-FOR-CALL-TRACKER.bat

=====================================================

MAC/LINUX USERS:
----------------
1. Press Ctrl+C in backend terminal (stop old server)  
2. Run this command:
   cd backend && deno run --allow-all server.tsx

OR run:
   cd backend && ./RESTART-FOR-CALL-TRACKER.sh
